# Create Interactive Plotly

Transform a `ggplot2`/`plot2` object to an interactive plot using the
[Plotly R Open Source Graphing Library](https://plotly.com/r/).

## Usage

``` r
as_plotly(plot, ...)

plotly_style(plot, ...)
```

## Arguments

- plot:

  A `ggplot2` plot.

- ...:

  In case of `as_plotly()`: arguments to pass on to
  [`layout()`](https://rdrr.io/r/graphics/layout.html) to change the
  Plotly layout object

  In case of `plotly_style()`: arguments to pass on to
  [`style()`](https://rdrr.io/pkg/plotly/man/style.html) to change the
  Plotly style object
